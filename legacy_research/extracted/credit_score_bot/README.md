# Credit Score Bot

Telegram-бот для получения кредитного рейтинга через `universal-credit.com`.

---

## Структура проекта

```
credit_score_bot/
├── bot.py                  ← Точка входа, запускать именно его
├── requirements.txt        ← Зависимости
├── .env.template           ← Шаблон конфига (переименуйте в .env)
├── app/
│   ├── config.py           ← Загрузка настроек из .env
│   ├── keyboards.py        ← Inline-клавиатуры
│   ├── messages.py         ← Форматирование сообщений
│   ├── fsm/
│   │   └── states.py       ← Состояния для пошагового ввода
│   ├── handlers/
│   │   ├── form_handlers.py ← Пошаговое заполнение анкеты
│   │   └── ssn_handler.py   ← Обработка ввода SSN
│   └── services/
│       └── worker_pool.py   ← Управление пулом воркеров
└── cs_module/              ← Модуль парсера (положить рядом)
```

---

## Установка

**1. Установить зависимости:**
```bash
pip install -r requirements.txt
camoufox install
playwright install chromium
```

**2. Создать `.env` файл:**
```bash
cp .env.template .env
```
Открыть `.env` и заполнить `BOT_TOKEN`, `PROXY_USERNAME`, `PROXY_PASSWORD`.

**3. Положить `cs_module/` рядом с `bot.py`.**

---

## Запуск

```bash
python bot.py
```

---

## Команды бота

| Команда | Действие |
|---|---|
| `/check` | Начать пошаговое заполнение анкеты |

---

## Флоу работы

```
Пользователь → /check
    → Бот спрашивает имя
    → Бот спрашивает фамилию
    → Бот спрашивает адрес
    → Бот спрашивает город
    → Бот спрашивает штат
    → Бот спрашивает ZIP
    → Бот спрашивает дату рождения
    → Бот спрашивает доход
    → Бот показывает сводку + кнопки [✅ Отправить] [🔄 Начать заново]
    → Пользователь нажимает "Отправить"
    → Воркер обрабатывает запрос в фоне
    → Бот отправляет результат: ✅ Credit Score: 797
```
